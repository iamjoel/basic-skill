## 读书的过程
* 带着问题去读书。
* 不是每章都读，只读对自己有用的。
* 自己熟悉的快速的读，陌生的，有用的，细致的读。
* 将书的内容进行整理，归档。

## Tip
* 理解过的东西越多，理解新的东西就越快。
